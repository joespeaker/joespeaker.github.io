SloVerb

Overview

SloVerb allows users to paste a YouTube link to a song and adjust three parameters:

Features

  • User Interface: Simple and Intuitive UI allows users to immediateatly understand the web app
  • Dark Mode: Toggle between light and dark modes for extra customization
  • Slider Controls: Allow fine adjustments to the output file
  • Submission: Submit the form to process data and generate your file

Installation

  1. Clone this repository to your local machine.
  2. Install required dependencies using pip install -r requirements.txt.
  3. Run python app.py to start the Flask application.
  4. Open your web browser and go to http://localhost:5000 to access the SloVerb app.

Technologies

  • Python
  • Flask
  • HTML
  • CSS

LANY Lyrics

This is an API I created that gives you a random line of a random LANY song.

You can use this API to make a website, app, or anything else you can think of!

To use this API, you can use the following URL: https://python.joespeaker.repl.co/api/lyric

You can also use the following URL to get a random line from a specific song: https://python.joespeaker.repl.co/api/lyric?song=ILYSB
